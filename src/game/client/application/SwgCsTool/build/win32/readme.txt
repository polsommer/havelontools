Please note:  The .vcproj file for this project is located in the src/win32 directory, due to a limitation inherent in the version of C# in VS.Net 7.1.  All source files in that version must be located in the same directory as the .vcproj file, or a subdirectory of it.

This may be revisited if this project is ever ported to a newer version, and if the newer version does not have this limitation.
